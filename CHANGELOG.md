# Changelog

All notable changes to this project will be documented in this file.

- This release supports **Bevy 0.18**.

## [0.6.0]

- This release supports **Bevy 0.17**.

### Changed

- Updated to Bevy 0.17.

## [0.5.1]

- This release supports **Bevy 0.16**.

### Fixed

- Fixed a panic on WASM by updating the version of `bevy_async_task`.

## [0.5.0]

- This release supports **Bevy 0.16**.

### Changed

- Updated to Bevy 0.16.

## [0.4.0]

- This release supports **Bevy 0.15**.

### Changed

- Updated `bevy_async_task` to 0.5.

## [0.3.1]

- This release supports **Bevy 0.15**.

### Fixed

- Elided some unnecessary lifetimes.

## [0.3.0]

- This release supports **Bevy 0.15**.

### Changed

- Updated to Bevy 0.15.

## [0.2.0]

- This release supports **Bevy 0.14**.

### Changed

- Updated to Bevy 0.14.

## [0.1.0]

- This release supports **Bevy 0.14**.
- Initial release. Crate was renamed to `bevy_key_rotation` from `bevy-key-rotation` and republished.
